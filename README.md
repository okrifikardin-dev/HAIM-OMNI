# 🐺 HAIM OMNI - Advanced Pentesting Framework
![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![License](https://img.shields.io/badge/License-GPLv3-red.svg)

**HAIM OMNI** هي أداة متطورة تم تطويرها في الجزائر (عين الحجل) مخصصة لاختبار الاختراق وتوليد الروابط العالمية والهجمات المتقدمة.

## 🚀 المميزات (Features)
* **Web Architect:** بناء صفحات مزورة بدقة عالية.
* **Global Tunneling:** توليد روابط عالمية عبر Cloudflare/Serveo.
* **Payload Generator:** إنشاء حمولات مخصصة.
* **Security Scanning:** فحص الثغرات (SQLMap/Hydra integration).

```bash
git clone [https://github.com/okrifikardin-dev/HAIM-OMNI.git](https://github.com/okrifikardin-dev/HAIM-OMNI.git)
